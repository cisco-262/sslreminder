# SSL Reminder Website

Static marketing and support website for SSL Reminder.

## Pages

- `/` - product homepage
- `/privacy/` - privacy policy
- `/support/` - support page
- `/about/` - product and OpsHome context

## GitHub Pages

This folder is designed for GitHub Pages hosting with the custom domain:

```text
ssl.opshome.run
```

The `CNAME` file is already included.

## App Store URL

The homepage App Store buttons point to the live SSL Reminder listing:

```text
https://apps.apple.com/us/app/ssl-reminder/id6786519365
```
