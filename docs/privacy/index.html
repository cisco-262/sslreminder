SSL Reminder

Privacy Policy

SSL Reminder is designed to help users monitor SSL/TLS certificate expiry dates and receive notifications while collecting as little information as necessary.

1. Overview

SSL Reminder helps users monitor SSL/TLS certificate expiry dates and receive notifications. We aim to collect as little information as necessary to provide this service.

2. Data We Use

When cloud monitoring is enabled, SSL Reminder may process:

  • The certificate targets you add, including host and port
  • An anonymous device identifier
  • Apple push notification token
  • Certificate status summaries and temporary detailed check records

3. Data We Do Not Collect

SSL Reminder does not collect:

  • Server passwords
  • Certificate private keys
  • DNS credentials
  • Website content
  • Personal identity information for account registration

4. Retention

Detailed check records are retained for 48 hours. After that, only the latest status summary and latest successful check summary are retained.

Targets inactive for more than 90 days are marked inactive. Inactive cloud data is deleted after 30 additional days unless the app becomes active again.

Users may delete cloud data at any time from within the app.

5. No Account Requirement

SSL Reminder does not require account registration.

6. Contact

For privacy-related questions, please contact OpsHome through the official support channels.
