关于 SSL Reminder

SSL Reminder 是 OpsHome 构建的一款专注 SSL 证书到期提醒的 iPhone 应用。

它是什么

SSL Reminder 帮助开发者、站长和 homelab 用户监控公开 SSL/TLS 证书的到期时间。它刻意保持轻量：添加证书目标、查看状态，并在到期前收到推送提醒。

为什么需要它

证书到期问题平时不显眼，但一旦发生就可能影响网站、API、控制台或个人服务。SSL Reminder 提供低成本的可见性，避免完全依赖人工记忆。

OpsHome 产品体系

OpsHome 构建面向实际运维可见性的工具。SSL Reminder 专注证书到期提醒；OpsHome NOC 面向更完整的正常运行时间、私有服务、Docker Probe、Synology、Proxmox、Linux 主机、Docker 容器和 homelab 运维。
